# AI-Powered Customer Retention Intelligence Platform

## Overview

This project is an end-to-end machine learning application that predicts customer churn and provides business intelligence through interactive dashboards.

## Features

- Customer Churn Prediction
- Risk Scoring Engine
- AI Recommendation Engine
- Revenue Intelligence
- Executive Dashboard
- Reports & Downloads

## Technologies

- Python
- Streamlit
- Scikit-learn
- XGBoost
- Pandas
- Plotly
